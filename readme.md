# Fullstack chat app
## Angular for Front and Nodejs for Back

### Install + Dev servers start
1. Clone repository
`git clone https://github.com/rjsnhk/INT219-LIVE-CHAT.git`
2. Install node_modules
`npm install`
3. Start front Angular
`cd front-angular`
`ng server`
3. Start front server
`cd sever-node`
`npm run build`
`node ./build/index.js`
